## üåê **Technical Details**

### **Tech Stack:**
- **Frontend:** Next.js 14, React, TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, Prisma ORM
- **Database:** PostgreSQL (Supabase)
- **Authentication:** NextAuth.js
- **Deployment:** Vercel

### **Installation & Architecture:**

**Cookie Banner Setup:**
1. User customizes banner in visual builder
2. Clicks "Generate Code"
3. Copies HTML/JavaScript snippet
4. Pastes into website (before `</body>` tag)
5. Banner appears automatically

**Server-Side Analytics Setup:**
1. Install server container (simple tag manager setup)
2. Configure data governance rules (anonymization, processing)
3. Connect to analytics platforms (Google Analytics, Facebook Pixel, etc.)
4. All measurement flows through YOUR server container
5. Data is processed and governed BEFORE reaching analytics platforms
6. Works with Google Consent Mode V2
7. Get 100% traffic data even when users reject cookies

### **Platform Support:**
- WordPress (via plugin or manual code)
- Shopify (via theme code injection)
- Webflow (via custom code block)
- Custom HTML/JavaScript sites
- Any platform that supports HTML/JS
